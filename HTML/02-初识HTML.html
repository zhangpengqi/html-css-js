初识HTML

简介


HTML 俗称"网页", 专业术语: 超文本标记语言(Hyper Text Markup Language), 由各种 标签 和 文本 组成的结构.

例如: 我们平时上网所看到的 百度, 淘宝 等都是需要HTML支持的, 否则我们就看不到页面啦

组成


HTML标签格式基本分为:
  • 闭合标签
  • 空标签
闭合标签

闭合标签, 又称双标签, 有开始, 有结束即为闭合, 都是成对出现的

格式: <开始标签> 文本 </结束标签>

例如: <b></b>标签, <i></i>标签, <p></p>标签

空标签

空标签, 又称单标签

格式: <开始标签>

严格格式: <开始标签 />

例如: <br>标签, <hr>标签, <input>标签

解释:
  • 严格写法通常是指XHTML
  • XHTML中, 空标签必须要通过 / 闭合
  • HTML中, 空标签对于加不加 / 都能兼容
  • HTML 和 XHTML 是网页两种不同的标准, 目前大家只需专注HTML即可
